# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming experience for everyone, regardless of background, experience level, or identity.

## Our Standards

**Positive behaviour includes:**

- Being respectful and constructive in discussions
- Welcoming contributions from people at all skill levels
- Providing helpful, actionable feedback on issues and PRs
- Focusing on what is best for the project and the security research community

**Unacceptable behaviour includes:**

- Using this project to conduct unauthorized investigations against real people or organizations
- Sharing output from unauthorized scans or investigations in issues, PRs, or discussions
- Harassment, personal attacks, or discriminatory language
- Publishing others' private information without permission

## Ethical Use Requirement

By contributing to or using this project you agree to:

1. Only investigate targets for which you have **explicit written authorization**
2. Comply with all applicable local, national, and international laws
3. Not use this tool to harass, stalk, dox, or harm any individual or organization
4. Responsibly disclose any security issues found via authorized testing

Violations of this requirement may result in removal from the project and may be reported to relevant authorities.

## Enforcement

Instances of unacceptable behaviour may be reported to the project maintainers by opening a private GitHub Security Advisory or via the contact information in SECURITY.md.

Maintainers will review all reports and respond as they deem appropriate, up to and including permanent removal from the project.
